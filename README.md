## How to setup

>**Create new directory** `Optional Step`
```console
 mkdir 'DTCID
```
```console
 cd 'DTCID
```

>**Clone DTCID from github**
```console
 git clone "https://github.com/hexdee606/DTCID.git"
```

>**Calling `.exe` file**
```console 
 cd DTCID
 cd Bin
 cd debug
 DTCID.exe
```
